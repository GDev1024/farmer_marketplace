Usage Guidelines

Primary Buttons

  • • Use for the main action on a page
  • • Limit to one primary button per section
  • • Examples: "Add to Cart", "Checkout", "Submit"

Secondary Buttons

  • • Use for secondary actions
  • • Can have multiple per section
  • • Examples: "View Details", "Cancel", "Save Draft"

Text Buttons

  • • Use for subtle, low-priority actions
  • • Good for inline actions in lists
  • • Examples: "Edit", "Remove", "Learn More"

Icon Buttons

  • • Use when space is limited
  • • Always include accessible labels
  • • Examples: Favorites, Share, Close
