Body Text (Sans-serif)

Font Family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, 'Helvetica Neue', Arial, sans-serif

Lead Text - Important introductory text that draws attention and provides context for the content that follows. This text is larger and more prominent than regular body text.

Body Text - This is the standard body text used throughout the marketplace. It should be highly readable and comfortable for extended reading. The line height and spacing are optimized for readability across all device sizes.

Caption Text - Smaller text used for captions, metadata, and supplementary information that supports the main content.

Label Text - Used for form labels, navigation items, and other UI elements that need to be clear and scannable.

Line Heights

Line height variations for different content types

Tight leading (1.25) - Used for headings and titles where space is at a premium and readability is still maintained.

Snug leading (1.375) - Slightly more space than tight, good for subheadings and short paragraphs.

Normal leading (1.5) - Standard line height for most UI elements and shorter text blocks.

Relaxed leading (1.625) - More comfortable for reading longer paragraphs and body text content.

Loose leading (2.0) - Maximum spacing for special emphasis or when extra breathing room is needed.

Responsive Typography

Typography scales appropriately across device sizes

The typography system automatically adjusts font sizes based on screen size:

  • Mobile (< 640px): Smaller font sizes for better fit
  • Tablet (640px - 1023px): Medium font sizes
  • Desktop (1024px+): Full font sizes for optimal readability

Resize your browser window to see the typography scale in action.
